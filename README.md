The desktop companion is an animated avatar that walk and talk on your desktop,

**The companion have different features : 
- read google news.
- Alarm reminder.
- checkout last video uploaded by your favorite youtube channel.

**Available animations: 
- idle
- walk
- talk
- think
- fight
- jump
- sleep

1- run the "start-companion.bat" to verify if you have python and install necessarry libraries.
2- you can make your own companion:
*create new folder with your character "name" inside the "characters" folder.
*generate sprite sheet animations using AI models like (GPT image 2 / Nano banana 2)
*make the sprite sheet transparent.
*rename the generated transparent sprite sheets with the right animations (idle, talk, walk, sleep, think...)
